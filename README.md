# SleepFi Record – Demo Site

音楽レーベル「SleepFi Record」を想定したデモWebサイトです。
HTML / CSS / JavaScript を用いて、Webデザインおよびフロントエンド実装を行いました。

カラーリングや余白を抑えたシンプルな構成で、
コンテンツが見やすくなるよう意識しています。

## Demo
https://nade1011.github.io/sleepfi-record/

## Tech
- HTML5
- CSS3
- JavaScript（Vanilla JS）
- レスポンシブ対応（PC / SP）

## Implementation
- data属性を用いたUI状態管理
- SPドロワーメニューの開閉制御
- オーディオプレーヤーの再生／停止制御
- position: sticky を用いた操作性を意識したUI
